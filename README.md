# Flutter Email Validation Field

This is a simple Flutter example that uses TextEditingController and TextFormField to take and validate email input.

## Features
- Uses TextEditingController to manage text input
- Email input field
- Basic validation (field cannot be empty)

## Technologies
- Flutter
- Dart

## How It Works
- TextEditingController is used to control and access user input
- The validator checks if the field is empty
- If empty, an error message is shown

## How to Run
1. Create a Flutter project
2. Add this code inside your widget
3. Run `flutter run`

## Usage
Enter an email in the field. If left empty, an error message will appear.
